<h2>Web (Next.js)</h2>
<p>UI for KPIs, Popular Products, and Analytics (Revenue/Orders, RFM, Segments) with filters.</p>

<h3>Run</h3>
<pre><code>npm run web:dev</code></pre>

<h3>Env</h3>
<ul>
  <li><code>NEXT_PUBLIC_API_BASE</code> – e.g. http://localhost:3001</li>
  <li><code>NEXT_PUBLIC_STORE_ID</code> – existing Store id</li>
</ul>
